zae-engine documentation!
======================================
zae-engine은 딥러닝 프로젝트의 quick start를 돕기 위한 파이썬 라이브러리입니다.
이 문서는 pre-release 버전이며, 버전관리가 되고 있지 않습니다.

아래의 명령어를 통해 설치 확인 및 각 기능을 지원 예정입니다.

.. toctree::
   :maxdepth: 2
   :caption: Command:

.. code-block:: console

   zae hello   >> 안녕하세요
   zae example >> 예제 파일 생성
   zae tree    >> 패키지 구조 출력
